# 饮料瓶收集合成游戏 — 设计规格说明书

> 版本: V0 (本地原型)  
> 日期: 2026-05-23  
> 平台: 微信小程序 (后续上云)

---

## 1. 游戏概述

一款以**自动贩卖机为核心锚点**的像素风格饮料瓶收集合成游戏。玩家通过金币在贩卖机抽卡获取瓶子，收集不同稀有度和主题的饮料瓶，通过三合一合成升级稀有度，最终完成全图鉴收集。支持好友交换和交易市场（V1/V2）。

### 核心循环

```
签到/分享/卖出 → 获得金币 → 贩卖机抽卡 → 收集瓶子 → 三合一合成/配方合成 → 解锁新贩卖机 → 更多抽卡
```

---

## 2. 技术架构

- **框架**: 微信小程序原生 + TypeScript
- **渲染**: Canvas 2D（像素风格高效绘制）
- **后端**: 微信云开发 CloudBase（V1启用）
- **原型策略**: V0 全本地数据，核心逻辑抽离为 Service 层，通过 DataAdapter 接口抽象，后续切换云适配器即可上云

### 演进路径

| 阶段 | 内容 |
|------|------|
| V0 本地原型 | 全本地存储，验证核心玩法趣味性 |
| V1 云端上线 | 切换 DataAdapter，接入云数据库，微信登录+分享，真实季节轮换 |
| V2 社交完整 | 交易市场、排行榜、活动系统、联动活动 |

## 3. 数据模型

### 3.1 瓶子 (Bottle)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识，如 "cola_001" |
| name | string | 瓶身名称 |
| rarity | 1-5 | 稀有度：⭐⭐普通 ⭐⭐稀有 ⭐⭐⭐珍贵 ⭐⭐⭐⭐史诗 ⭐⭐⭐⭐⭐传说 |
| category | enum | 分类：coffee / soda / tea / sugarfree / juice / energy |
| limited | boolean | 是否限定瓶 |
| season | string\|null | 限定季节标签：spring / summer / autumn / winter / holiday / collab |
| sprite | string | 像素素材路径 |
| synthesizable | boolean | 是否可合成获得 |
| recipe | {a, b, c}[] | 配方合成表（稀有品专用） |

### 3.2 贩卖机 (VendingMachine)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 如 "vm_soda" |
| theme | enum | 主题分类 |
| unlockType | enum | 解锁方式：coin / synth / task |
| unlockCost | number | 金币价格（coin类） |
| pullCost | number | 单抽金币消耗 |
| pool | PoolConfig | 抽取概率配置 (含各稀有度概率 + 保底数) |
| sprite | string | 贩卖机像素素材 |

### 3.3 玩家数据 (PlayerData)

| 字段 | 类型 | 说明 |
|------|------|------|
| coins | number | 金币余额 |
| bottles | Map<id, count> | 持有瓶子及数量 |
| machines | string[] | 已解锁贩卖机ID列表 |
| collectionProgress | {total, byRarity, limitedCount} | 图鉴收集进度 |
| stageBlindBox | {collected, threshold} | 阶段盲抽进度 |

## 4. 概率配置

| 稀有度 | 基础概率 | 保底机制 |
|--------|---------|---------|
| ⭐⭐⭐⭐⭐ 传说 | 0.5% | 200抽保底 |
| ⭐⭐⭐⭐ 史诗 | 3% | 50抽保底 |
| ⭐⭐⭐ 珍贵 | 12% | 15抽保底 |
| ⭐⭐ 稀有 | 30% | - |
| ⭐ 普通 | 54.5% | - |

- 再来一瓶概率：8%（独立事件）
- 阶段盲抽：每收集30种新瓶子触发1次（保底 ⭐⭐⭐+）

## 5. 限定瓶系统

限定瓶具有 `limited=true` 标记，是游戏中稀缺性最高的收集目标。

### V0 策略
- 限定瓶常驻可见（方便原型测试），放入第4台贩卖机「限定·珍藏」
- 解锁方式：金币购买（定价高于普通贩卖机）
- 约10种限定瓶示例：春季2种 / 夏季2种 / 秋季2种 / 冬季2种 / 节日2种
- 奖池中限定瓶出现率高于同稀有度普通瓶

### V1 升级
- 真实季节轮换：春/夏/秋/冬按现实时间切换奖池
- 节日限定：春节/中秋/圣诞等限时7天
- 联动限定：与IP合作专属瓶

## 7. 合成系统

### 7.1 三合一升级（常规）
- 3个同稀有度瓶子 → 随机高一级稀有度瓶子
- 限定瓶不可作为合成材料（保护稀缺性）
- 费用：按目标稀有度递增 (10/30/80/200 金币)

### 7.2 配方合成（V1）
- 特定组合解锁唯一珍藏品
- 配方提示在图鉴中暗藏线索

### 7.3 随机惊喜
- **再来一瓶**: 每次合成/抽卡 8% 概率免费再获得一瓶
- **阶段盲抽**: 每收集30种新瓶子触发1次，保底 ⭐⭐⭐+（极小概率出限定瓶）

## 8. 经济系统

### 金币流入

| 来源 | 数量 | 频率 |
|------|------|------|
| 新手礼包 | 200 | 一次性 |
| 每日签到 | 20 | 每天 |
| 分享贩卖机 | 30 | 每天3次 |
| 卖出瓶子 | 2/5/15/50/200 | 按稀有度，无限制 |
| 收集成就 | 50~500 | 里程碑 |

### 金币流出

| 用途 | 费用 |
|------|------|
| 单抽 | 10 |
| 十连抽 | 90 |
| 三合一合成 | 10/30/80/200 |
| 解锁贩卖机 | 50~300（限定贩卖机定价偏高） |
| 交易市场手续费 (V2) | 5% |
